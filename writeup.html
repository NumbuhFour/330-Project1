The Cleverly-Titled Space-esque Hyphen-heavy Game!

Overview

This is a small game written for 330-Rich Media Web App Dev

The concept is a platformer jumping from gravitational bodies.
The dotted lines mark the influence range of each planet. Moving within them will cause you to be affected by the planet's gravity.

The goal will be to collect the gold on each of the planets while avoiding dangers, maintaining fuel, and not drifting off into the endless void of space. Then to reach the warp gate at the end of the map to move on to the next.

Details

The skeleton of the code behind this is derived from the Blastem projected provided by my professor, Sean Boyle.
The game works by applying basic physics (velocity & whatnot) to the player and spinning the canvas around using canvas tranformations. Each time velocity is applied to the player through movement (or slowing) a bit of fuel is taken away based on how much velocity is added.
Each level is loaded by reading in a js-object-literal written in the loader.js file. This could easily be converted over to JSON later but for now I didn't want to implement any external-file reading since GitHub was not making resource loading easy.
Also, the game is kind of hard. But you don't need all the coins.

Issues

The biggest issue I had is getting some features to work through GitHub. Namely resource loading since the way provided by the course doesn't work due to cross-origin sharing policies. Everything works locally, but unfortunately I'll need to migrate things back to gibson to make them work online.
