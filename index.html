The Cleverly-Titled Space-esque Hyphen-heavy Game!

Controls

  • Left/Right: Rotate (while in space), Walk (while on surface)
  • Up/Down: Thrust
  • Space: Slow ship
  • S: Toggle music

Overview

This is a small game written for 330-Rich Media Web App Dev

The concept is a platformer jumping from gravitational bodies.
The dotted lines mark the influence range of each planet. Moving within them will cause you to be affected by the planet's gravity.

The goal (to be implemented) will be to collect the gold on each of the planets while avoiding dangers, maintaining fuel, and not drifting off into the endless void of space. Then to reach the warp gate at the end of the map to move on to the next.

Details

The skeleton of the code behind this is derived from the Blastem projected provided by my professor, Sean Boyle.
The game works by applying basic physics (velocity & whatnot) to the player and spinning the canvas around using canvas tranformations. Each time velocity is applied to the player through movement a bit of fuel is taken away based on how much velocity is added.
